## *** CORRECTION TO EASY SAVE DOS WEDGE ***

ROY Baker reports that the following correction to the wedge program published in the August newsletter is required.

Prior to Step 2. add Step 1a:

- 1a. Find the start of variables

```PRINT PEEK (45) + 256 * PEEK (46)```

Add 860 to the result. (New start place)

```POKE 46, (New start) /256 POKE 45, (New start) - 256 * PEEK (46)```

Now proceed to Step 2.
