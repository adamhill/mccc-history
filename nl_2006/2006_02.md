## AmiKit 1.0.3

24 December, 2005 AmiKit is the compilation of the best Amiga (free/shareware) programs around. To be brief, AmiKit is all you need if you want to experience a highend Classic Amiga system. It contains many expansions and add-ons (more than 20.000 files are included in AmiKit!) that turn the default AmigaOS installation into a completely renewed and fresh environment. This pre-configured package is designed for immediate use. Moreover, it contains an exciting surprise for the whole Amiga community, see the features!
